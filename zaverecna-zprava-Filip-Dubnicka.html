Popis mojej práce

Pre náš projekt som pomohol spraviť jeho konfiguráciu s pomocou Spring Boot platformy, vytvoril som REST vrstvu na strane backendu, pomocou technologie Angular frontend a s pomocou Semantic UI webové rozhranie.

REST vrstva

Rest vrstva je vytvorená na základe Spring frameworku, ponúka volne dostupné REST rozhranie pre funkcie nášho projektu.

Frontend

Frontend som vybudoval na základe javascriptového frameworku Angular. Vytvoril som jeho konfiguráciu a následne v nom potrebné servicies pre prístup k REST rozhraniu a taktiež controllery k jednotlivým obrazovkám. Obrazovky som vyskladal s pomocou Semantic UI.
